# Vaibhav Patil — Portfolio

Personal portfolio site showcasing my journey as a software engineer building scalable systems, AI-powered applications, and full-stack products.

**Stack:** React · Vite · Tailwind CSS · Framer Motion · React Router

Live sections: Home, Timeline, Projects, Skills, Experience, Certifications, Achievements, Contact.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
